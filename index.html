    This is a forked project of the main BnS Skill Tree webside intended for the TW server.
    The only difference is the level cap, skill points acquired at each level, and skill point requirements for some skills. Warlock is also unavailable.

    This section won't be fully supported by me since I don't play on TW.
    But you are free to fork the project from the github repo yourself, make some changes and send pull requests.
